## 实施计划:生成 Qwen3-VL 自包含推理文件

### 设计原则(与官方 Qwen2.5-VL 一致)
1. **纯推理**:只含 generate/generate_with_bd3lm/prepare_inputs_labels_for_multimodal/forward(eval loss),不含训练专用逻辑(LogLinearNoise/block_diff_mask/_apply_bd3lm_noise_embedding/_compute_bd3lm_loss_embedding)
2. **不依赖 llava 包**:所有 llava 代码内联,trust_remote_code=True 即可用
3. **vision_tower/mm_projector 在 __init__ 直接构建**(作为普通子模块),权重由标准 HF from_pretrained 按 key 加载 —— 不需要 initialize_vision_modules/from_pretrained override/分文件加载
4. **从 transformers 导入 Qwen3VL 类**(用户有 >=4.57.0):Qwen3VLVisionModel/Qwen3VLVisionPatchMerger/Qwen3VLTextAttention 等,不重新实现骨干

### 新建 3 个文件(放在 scripts/diffusionvl_prepare/ 下)

#### 1. configuration_diffusionvl_qwen3_vl.py(~60 行)
- DiffusionVLQwen3VLConfig(PretrainedConfig):model_type="diffusionvl_qwen3vl",is_composition=True
- 从 config.json 读 vision_config/text_config/enable_bd3lm/bd3lm_block_size/mask_token_id
- to_dict() 输出 vision_config/text_config
- 照搬官方的 kwargs.pop("text_config") 处理(避免 GenerationConfig 问题)

#### 2. modeling_diffusionvl_qwen3_vl.py(~1300 行)
合并来源:
- llava_diffusionvl_qwen3vl.py 的推理部分(去掉训练专用):Attention、TextModel(含 DeepStack 注入)、ForCausalLM 的 forward(eval loss)+ generate + generate_with_bd3lm + 采样辅助
- qwen3_vision_tower.py:LlavaQwen3VisionTower(简化,去掉 load_model/initialize 逻辑,直接作为子模块构建)
- qwen3_projector.py:LlavaQwen3Projector
- llava_arch.py 的 prepare_inputs_labels_for_multimodal + encode_images(Qwen3 分支)
- constants.py:3 个常量(IGNORE_INDEX/IMAGE_TOKEN_INDEX/DEFAULT_IMAGE_TOKEN)
- utils.py:rank0_print

关键改动:
- __init__ 直接构建 vision_tower/mm_projector(不调 initialize_vision_modules)
- 无 from_pretrained override:标准 HF 加载按 key 匹配
- 无 load_model/state_dict 手动加载:权重在 model.safetensors 里,HF 自动加载
- DeepStack 注入逻辑照搬(prefill + 逐块 cursor)
- image_processor 在 LlavaQwen3VisionTower.__init__ 里用 AutoImageProcessor.from_pretrained 加载

#### 3. processing_diffusionvl_qwen3_vl.py(~200 行)
- tokenizer_image_token(从 mm_utils 内联)
- DiffusionVLQwen3VLProcessor(ProcessorMixin):image_processor_class="Qwen2VLImageProcessor",tokenizer_class=("Qwen2Tokenizer","Qwen2TokenizerFast")
- __call__:处理 image + text,返回 BatchFeature(input_ids, attention_mask, pixel_values, image_grid_thw)
- build_conversation_input_ids:ChatML 格式 prompt 构建

### config.json 修改
给训练 checkpoint 的 config.json 加 auto_map:
```json
"auto_map": {
    "AutoConfig": "configuration_diffusionvl_qwen3_vl.DiffusionVLQwen3VLConfig",
    "AutoModelForCausalLM": "modeling_diffusionvl_qwen3_vl.DiffusionVLQwen3VLForConditionalGeneration",
    "AutoProcessor": "processing_diffusionvl_qwen3_vl.DiffusionVLQwen3VLProcessor"
}
```

### 使用方式
把 3 个 .py 拷到训练 checkpoint 目录,加 auto_map 到 config.json,确保有 preprocessor_config.json,然后:
```python
from transformers import AutoModelForCausalLM, AutoProcessor
model = AutoModelForCausalLM.from_pretrained("ckpt", trust_remote_code=True, torch_dtype="bfloat16")
processor = AutoProcessor.from_pretrained("ckpt", trust_remote_code=True)
```
